order

Permissible values: [-∞; +∞]
Defalut value: 0
    .flex-container {
        display: flex;
        
        .flex-element-1 {
            order: 1; // Первый элемент будет перемещен в конец

        .flex-element-3 {
            order: -1; // Третий элемент будет перемещен в самое начало, т.к по умолчанию у всех элементов свойство order: 0
        }

        .flex-element-5 {
            order: 2; // Теперь пятый элемент будет в самом конце, т.к индекс его свойство order: 2 > свойство order: 1 у первого элемента 
        }
    }
    // Данный пример можно сопаставить с координатной прямой X.
    // Если число больше, то оно будет дальше по направление возрастания, в нашем случае это flex-direction: row
    // то есть от -∞ к +∞, соотсетственно элементы, у которых значение значение свойства order будет больше, те элементы будут стремится
    // занять позицию правее относительно своих "братьев" на одном уровне, ближе к +∞, аналогично и с наименьшим значение свойства order,
    // элементы будут стремиться занять позицию левее, ближе к -∞.
